# Suggestions Library

This folder stores **constructional (constructive, implementation-ready) suggestions** for TU-VM, with a strong focus on:

- avoiding reinvention,
- using proven frameworks,
- improving day-to-day contributor workflows, and
- building a sustainable community contribution system.

## Current status

No historical suggestions were found in this repository before this folder was created.  
These files establish the first baseline set of proposals.

## How to use this folder

- Treat each file as a living proposal.
- Keep old decisions in place; add decision updates instead of deleting context.
- Reference real repository paths/scripts when proposing changes.
- Prefer incremental rollout to reduce operational risk.
