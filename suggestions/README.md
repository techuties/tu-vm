# Suggestions Archive

This folder stores **constructional suggestions** for the TechUties VM website and community ecosystem.

Goals:
- Keep a historical record of ideas so we do not re-invent the wheel.
- Propose practical frameworks and tools that integrate with the current stack.
- Prioritize community-first workflows that are easy to operate day to day.

## Current suggestion set

1. `website-platform-frameworks.md`  
   Recommended website architecture, framework choices, and integration with the current Docker/Nginx setup.
2. `website-community-and-governance.md`  
   Community features, moderation model, contributor roles, and trust/safety guardrails.
3. `website-contributor-tooling.md`  
   Day-to-day tooling to speed up maintenance, triage, docs, and release workflows.

## Historical tracking

When adding future ideas:
- Prefer updating existing files if a section already covers the topic.
- Add a dated subsection (for example: `## 2026-03-12 updates`) rather than creating many small files.
- Keep each recommendation tied to a concrete outcome, owner, and effort level.
